Cats vs Dogs

A computer vision project that identifies between cats and dogs in a pictures

A project done to familiarize myself with the fascinating world of computer vision. In this project I, along with a team member built an application using OpenCV and Tensorflow that differentiates between cats and dogs in a picture. We participated in a Kaggle competition with 20 teams and managed to grab 1st position with an accuracy of 94%. The details of how we went about building the application can be found below
