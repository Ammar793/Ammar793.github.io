Rickshaw vs Donkey Cart racing game

A basic demo of a 3D game built with the purpose of familiarising myself with the concepts of 3d modeling/texturing with Maya lt as well as game developing with Unity 3D. Use the mouse and arrow keys to move the donkey cart and beat the Rickshaw in a 4 lap race around cramped roads. Enjoy! (It might not work on Chrome)
