4x4x4 LED Cube

LED Cube designed to light up with different designs

Using an arduino microcontroller I designed a 4x4x4 LED cube that lights up in different patterns. To achieve this I used 64 LEDs which I soldered togethor to create a multiplex which needed 16 inputs to light up all the LEDs instead of 64.
