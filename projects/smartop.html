SmartOp

Android application to aid deaf users in making phone calls to businesses

SmartOp is a smartphone application built with the aim to allow hard of hearing people to communicate with businesses over the phone. It uses audio recognition to convert audio over the telephone to textual information which you can easily read and then respond to by using a text-to-speech interface. There is no middle man and we are aiming to make the user interface easy to use by including functionality like saving personal information and pre-saved greeting messages to enable the users to be able to converse with operators easily.

For this project my team made sure that the users themselves were involved in the design process from the very beginning, so we had users with varying levels of deafness participate in usability testing procedures throughout the development life cycle.
